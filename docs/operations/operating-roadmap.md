# Operating Roadmap

## 1. Time Horizon

This roadmap starts on **April 5, 2026** and runs through **March 31, 2030**.

## 2. Quarter-By-Quarter Plan

### 2026 Q2: April to June 2026

- freeze company thesis and product split
- convert MVP into evidence-platform requirements
- set up requirements traceability, risk register, and release checklist
- define session contract and clinic dashboard requirements
- begin workflow feasibility study planning

### 2026 Q3: July to September 2026

- deploy identity, consent, audit, and QC workflows
- finish study A workflow feasibility work
- qualify the clinic capture rig
- finish pilot protocol, case-report forms, and label workflow

### 2026 Q4: October to December 2026

- launch the first prospective single-site pilot
- begin proprietary data collection
- build speech-first baseline models
- ship investigational dashboard to pilot staff

### 2027 Q1: January to March 2027

- complete the first interim pilot readout
- improve calibration and QC handling
- begin home substudy enrollment for selected participants
- finalize the provider deployment playbook for the pilot site

### 2027 Q2: April to June 2027

- expand cohort collection
- add structured-task features to the clinic model
- quantify subgroup behavior by language and age band
- validate home baseline completion logic

### 2027 Q3: July to September 2027

- push toward 150-200 participant labeled dataset target
- complete expanded Singapore pilot readout
- decide whether facial analysis stays research-only or advances to formal validation
- prepare external validation package

### 2027 Q4 To 2028 Q2: October 2027 to June 2028

- start multi-site validation
- freeze candidate model specification
- formalize home escalation rules and provider review workflow
- harden dashboard reporting and audit exports

### 2028 Q3 To 2029 Q1: July 2028 to March 2029

- complete locked-model external validation
- finish English and Mandarin production readiness
- run health-economic and workflow studies
- complete registration planning package

### 2029 Q2 To 2030 Q1: April 2029 to March 2030

- submit or prepare to submit the first regulated provider product if evidence supports it
- convert at least one pilot into paid deployment
- launch provider-managed home monitoring as an extension offering
- establish post-market monitoring and controlled model update cadence

## 3. Workstream Owners

- Clinical and partnerships: founder plus clinical advisor
- Product and workflow: product lead
- Platform and dashboard: full-stack engineer
- ML and data: ML engineer
- Data operations and studies: clinical research manager
- Regulatory and quality: fractional regulatory lead

## 4. Company-Level KPIs

- pilot sites contracted
- participants enrolled
- usable sessions collected
- QC pass rate
- sensitivity, specificity, and ROC-AUC on locked data
- home baseline completion rate
- paid site conversion rate

## 5. Immediate Next 12 Weeks

- finalize product and data contracts
- set up the design-control template system
- define the pilot protocol and labeling workflow
- build a clinic dashboard specification
- inventory benchmark datasets and access constraints
- stand up reproducible feature extraction and model versioning

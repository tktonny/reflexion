# 版本 2（http）· 安装包 · 安装与使用

**一句话**：真机安装 `reflexion-v2-http.apk`，**回合制"按住说话"**和 Aria 对话，**不需要任何服务器**（Qwen key 已内置，联网即用）。

- 安装包：`dist-apks/reflexion-v2-http.apk`（约 52MB，arm64-v8a，Android 7.0+，debug 签名）
- 对话方式：**回合制**——你按键说话 → 松开发送 → Aria 回一段 → 再轮到你。每一轮内部是 录音 → 语音识别(ASR) → 对话(LLM) → 语音合成(TTS)。
- 适合：不方便配麦克风流的环境、或先验证"能装能跑能出判断"。真机上想要更自然的全双工体验请用 v3。

---

## 一、安装

### USB + adb（推荐）
```bash
adb install -r dist-apks/reflexion-v2-http.apk
```
出现 `Success` 即成功（前提：手机已开「开发者选项 → USB 调试」）。

### 侧载（无电脑）
把 `reflexion-v2-http.apk` 传到手机 → 文件管理器点开 → 允许"安装未知应用" → 安装。被 Play Protect 拦就选"仍要安装"。

> ⚠️ 与 v3 包名相同（`com.reflexion.mirror`），**不能与 v3 同时安装**，后装的会覆盖先装的。

### 授权
首次进对话，全部允许：🎤 **麦克风**（必需）、📷 **相机**（多模态筛查用）。

---

## 二、进入"每日检查"

1. 打开 App，等启动自检跑完（几秒）。
2. 因为没配后台，会停在**配对页**——**不用管配对**，直接点最下面金色按钮 **「开始每日检查(演示)→」**。
3. 进入 **「Reflexion 检查 · 测试」** 界面。顶部"版本"应显示 **http**。

---

## 三、对话（v2 的关键：按住说话）

1. 点 **「开始对话」**，授权麦克风/相机。Aria 先开场（你会听到语音 + 看到字幕）。
2. 轮到你说时：
   - 点绿色 **「🎤 开始说话」** → 开始录音。
   - 说完点 **「发送（结束说话）」** → App 把这段发去识别 + 交给 Aria。
   - 稍候 Aria 语音 + 字幕回复。
3. 如此一问一答，走完 4 个阶段。
4. 结束：
   - Aria 说出告别语 → **自动结束并生成判断**；
   - 或任何时候点 **「结束并评估」** 手动结束。
5. 出判断卡片：分级、风险分、发现、支持/反对风险证据、**视觉观察**。

> 若某一轮卡在"处理中"：单轮有 45s 超时保护，超时会自行退出 spinner，可重说一轮。

---

## 四、确认与排错

- **确认版本/硬件**：对话界面点 **「🔧 硬件自检」**，`configuredMode` 应为 `http`。
- **点了"开始说话"没反应**：确认已授予麦克风权限（系统设置 → 应用 → Reflexion Mirror → 权限）。
- **一直转圈/不回话**：多为网络到 Qwen 不通；换更稳的网络重试。单轮 45s 超时会自动解除。
- **装不上**：确认是 arm64 真机、Android 7+；若之前装过 v3，先 `adb uninstall com.reflexion.mirror` 再装。

---

## 五、v2 vs v3

| | v2 (http) | v3 (ws) |
|--|-----------|---------|
| 说话 | 按住说话，回合制 | 直接说，全双工流式 |
| 延迟 | 每轮有识别/合成延迟 | 近实时，可打断 |
| 依赖 | 无服务器 | 无服务器（直连 Qwen） |
| 模拟器 | 可跑 | 麦克风 HAL 不工作，需真机 |

真机体验优先选 **v3**；v2 作为"稳妥回退"和一键验证。
